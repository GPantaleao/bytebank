# Bem-vindo ao ByteBank UI  🐷

O **ByteBank UI** é o sistema de design e biblioteca de componentes desenvolvida para a plataforma financeira ByteBank. 
Nosso objetivo é garantir consistência visual, acessibilidade e agilidade na construção das interfaces de usuário.

##  Objetivo deste Storybook

Este Storybook é o nosso guia oficial de interface. Ele centraliza todos os componentes e padrões visuais do projeto, permitindo que desenvolvedores, designers e Product Managers possam:

1. **Explorar componentes:** Ver todos os botões, inputs, modais e cartões disponíveis no projeto.
2. **Entender os comportamentos:** Testar os estados interativos (hover, focus, disabled, loading) sem precisar rodar o sistema inteiro.
3. **Validar regras de negócio:** Através da aba *Docs* e dos testes de interação automatizados (cenários).
4. **Consultar Tokens:** Ter acesso rápido à paleta de cores oficial e à tipografia.

##  Nossa Arquitetura Visual (Atomic Design)

Seguimos a metodologia do **Atomic Design** para organizar nossos componentes de forma escalável:
